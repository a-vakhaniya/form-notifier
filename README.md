# 📬 Form Notifier — уведомления с сайта в Telegram и Google Sheets

## Что делает
Получает данные с формы сайта (имя, телефон, сообщение)
и одновременно отправляет их в Telegram-бот и записывает в Google Sheets.

## Схема работы
Webhook → Telegram Bot
        ↘ Google Sheets

## Стек
- Make.com (автоматизация)
- Telegram Bot API
- Google Sheets

## Зачем
Мгновенные уведомления о новых заявках с сайта.
Все заявки автоматически сохраняются в таблицу — история не теряется.
